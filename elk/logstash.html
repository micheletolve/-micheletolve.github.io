Logstash

Logstash

Pipeline con la funzione di raccogliere dati da diverse fonti e mandarle ad elasticsearch.

Config

Scaricare logstash all'indirizzo https://www.elastic.co/downloads/logstash

Preparare il file logstash.conf per seguire la pipeline Beats -> Logstash -> Elasticsearch.

Come input mi metto in ascolto sulla porta 5044 per ricevere da filebeats.

input {
  beats {
    port => 5044
  }
}

Tramite grok è possibile ricavare la struttura di dati non strutturati e modificare i dati on the fly. Nel nostro caso suddividiamo riga per riga con il pattern descritto nello snippet seguente. Inoltra ignoriamo i messaggi che sono più vecchi di un certo numero di giorni (poichè una volta arrivati ad elastic i messaggi vengono smistati nei vari indici in base alla data, se dovesse arrivare un messaggio troppo vecchio potrebbe provare a scrivere in un indice che ormai è in sola lettura o eliminato, causando il blocco di tutta la pipeline)

filter {
  grok {
    match => { "message" => "%{LOGLEVEL:level} %{NOTSPACE:app} %{NOTSPACE:logger} %{NOTSPACE:class} %{NOTSPACE:method} %{NUMBER:row} %{TIMESTAMP_ISO8601:timestamp} %{NUMBER:thread} (?<message>(.|\r|\n)*)" }
    overwrite => [ "message" ]
      }
  date {
    match => ["timestamp", "ISO8601"]
  }
  ruby {
    init => "require 'time'"
    code => 'if LogStash::Timestamp.new(event.get("@timestamp")+172800) < ( LogStash::Timestamp.now)
              event.cancel
            end'
  }
  mutate {
    remove_field => [
                "[timestamp]",
                "[beat]",
                "[log]",
                    "[prospector]",
                    "[source]",
                    "[host]",
                "[tags]"
            ]
    }
}

In output mando ad elasticsearch costruendo un indice giornaliero eventualmente inserendo user e password

output {
  elasticsearch {
    hosts => ["http://localhost:9200"]
    index => "%{[@metadata][beat]}-%{[@metadata][version]}-%{+YYYY.MM.dd}"
    #user => "elastic"
    #password => "changeme"
  }
}

Run

Eseguire il comando

bin/logstash -f logstash.conf
